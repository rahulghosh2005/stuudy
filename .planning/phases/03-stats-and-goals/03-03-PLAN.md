**Props:**

interface DailyProgressBarProps {
  uid: string;

**Behavior:**
- If `dailyGoalEnabled === false`: return `null` (nothing rendered, so TimerPage progress bar area is completely hidden)
- On mount, query today's sessions using `getSessions(uid, startOfLocalToday)`
- "startOfLocalToday" = start of the user's local calendar day in their IANA timezone using `toZonedTime` from `date-fns-tz`
- Sum `durationMs` across all today's sessions → `todayMinutes`
- Render a thin bar below the timer showing: `Xh Ym / Zh goal — N%`

**Today's local midnight calculation** (from RESEARCH.md — must use user's timezone, not browser local):
```typescript
import { toZonedTime } from 'date-fns-tz';
import { startOfDay } from 'date-fns';

function getStartOfLocalToday(timezone: string): Date {
  const nowInUserTz = toZonedTime(new Date(), timezone);
  return startOfDay(nowInUserTz);
}
```

**Data fetch** (in useEffect on mount and when uid changes):
```typescript
useEffect(() => {
  if (!uid || !dailyGoalEnabled) return;
  const since = getStartOfLocalToday(timezone);
  getSessions(uid, since)
    .then(sessions => {
      const totalMs = sessions.reduce((acc, s) => acc + s.durationMs, 0);
      setTodayMinutes(Math.round(totalMs / 60000));
    })
    .catch(() => { /* non-blocking — progress bar just shows 0 */ });
}, [uid, timezone, dailyGoalEnabled]);
```

**Progress bar rendering:**
```tsx
const pct = dailyGoalMinutes > 0
  ? Math.min(100, Math.round((todayMinutes / dailyGoalMinutes) * 100))
  : 0;

const formatMin = (m: number) => {
  const h = Math.floor(m / 60);
  const rem = m % 60;
  return h > 0 ? `${h}h ${rem}m` : `${rem}m`;
};

return (
  <div style={{ width: '100%', maxWidth: 300, margin: '12px auto 0' }}>
    <div style={{ display: 'flex', justifyContent: 'space-between', marginBottom: 4 }}>
      <span style={{ color: '#888', fontSize: 11 }}>
        {formatMin(todayMinutes)} / {formatMin(dailyGoalMinutes)} goal
      </span>
      <span style={{ color: pct >= 100 ? '#fc4c02' : '#888', fontSize: 11, fontWeight: pct >= 100 ? 700 : 400 }}>
        {pct}%
      </span>
    </div>
    <div style={{ background: '#1a1a1a', borderRadius: 3, height: 4 }}>
      <div style={{
        background: '#fc4c02',
        borderRadius: 3,
        height: 4,
        width: `${pct}%`,
        transition: 'width 0.3s ease',
      }} />
    </div>
  </div>
);
```

Per CONTEXT.md: "thin bar below the timer display showing `Xh Ym / Zh goal — N%`" — keep it subtle, not distracting during active timing.
